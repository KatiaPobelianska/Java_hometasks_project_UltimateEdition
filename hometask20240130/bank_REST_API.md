Клиент (Client) // сущность Client

Методы:

GET /clients
Описание: Получить список всех клиентов.
Ответ: Список сущностей Client.
Ошибки:
500 Internal Server Error: Внутренняя ошибка сервера.

GET /clients/{id}
Описание: Получить информацию о клиенте по его ID.
Параметры пути: id - Уникальный идентификатор клиента.
Ответ: Сущность Client.
Ошибки:
400 Bad Request: Некорректный формат ID клиента.
404 Not Found: Клиент с указанным ID не найден.
500 Internal Server Error: Внутренняя ошибка сервера.

POST /clients
Описание: Создать нового клиента.
Тело запроса: Данные нового клиента в формате JSON.
Ответ: Сущность Client созданного клиента.
Ошибки:
400 Bad Request: Некорректные данные клиента.
500 Internal Server Error: Внутренняя ошибка сервера.

PUT /clients/{id}
Описание: Обновить информацию о клиенте по его ID.
Параметры пути: id - Уникальный идентификатор клиента.
Тело запроса: Обновленные данные клиента в формате JSON.
Ответ: Сущность Client обновленного клиента.
Ошибки:
400 Bad Request: Некорректные данные клиента или ID.
404 Not Found: Клиент с указанным ID не найден.
500 Internal Server Error: Внутренняя ошибка сервера.

DELETE /clients/{id}
Описание: Удалить клиента по его ID.
Параметры пути: id - Уникальный идентификатор клиента.
Ответ: Подтверждение удаления.
Ошибки:
400 Bad Request: Некорректный формат ID клиента.
404 Not Found: Клиент с указанным ID не найден.
500 Internal Server Error: Внутренняя ошибка сервера.
